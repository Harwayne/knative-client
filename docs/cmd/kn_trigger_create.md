## kn trigger create

Create a Trigger and optionally an importer. If an Importer is created, then the created Trigger receives events exclusively from the created importer.

### Synopsis

Create a Trigger and optionally an importer. If an Importer is created, then the created Trigger receives events exclusively from the created importer.

```
kn trigger create NAME --subscriber K_SVC_NAME [flags]
```

### Examples

```

  # Create a trigger 'my-t' that sends all events sent to the 'default' Broker
  # to the Knative Service svc1.
  kn trigger create my-t --subscriber svc1

  # Create a trigger 'my-t' that sends all events sent to the 'other-b' Broker
  # to the Knative Service svc1.
  kn trigger create my-t --subscriber svc1 --broker other-b

  # Create a trigger 'my-t' that sends all events with CloudEvent attribute key
  # 'foo' with value 'bar' and key 'qux' with value 'garply' to the Knative
  # Service svc1.
  kn trigger create my-t --subscriber svc1 --filter 'foo=bar,qux=garply'

  # Create a trigger 'my-t' and a CronJobSource importer. The only events sent
  # to the subscriber, Knative Service svc1, are those generated by the created
  # importer.
  kn trigger create my-t --subscriber svc1 --importer CronJobSource

  # Create a trigger 'my-t' and an ApiServerSource importer. The only events
  # sent to the subscriber, Knative Service svc1, are those generated by the
  # created importer. The importer is created with its spec.mode = 'ref' and
  # spec.serviceAccountName = 'sa'.
  kn trigger create my-t \
    --subscriber svc1 \
    --importer apiserversources.sources.eventing.knative.dev \
    --importer-parameters 'mode=ref,serviceAccountName=sa'
```

### Options

```
      --async                                         Create trigger and don't wait for it to become ready.
      --broker string                                 Broker the Importer associates with. (default "default")
      --filter stringToString                         Filter attributes, expressed as a CSV. (default [])
      --force                                         Create importer forcefully, replaces existing importer if any.
  -h, --help                                          help for create
      --importer string                               Name of the Importer CRD to create pointing at this Trigger.
      --importer-event-types strings                  Comma separated list of event types.
      --importer-extension-overrides stringToString   CloudEvent extension attribute overrides. (default [])
      --importer-parameters stringToString            Parameters used in the spec of the created importer, expressed as a CSV. (default [])
      --importer-secret specField=secretName:key      Secret to inject into the spec as a SecretKeySelector. In the form specField=secretName:key. Which will set `spec.specField` to a SecretKeySelector. (default =:)
  -n, --namespace string                              List the requested object(s) in given namespace.
      --subscriber string                             Name of the Knative Service that is subscribing to this Trigger.
      --wait-timeout int                              Seconds to wait before giving up on waiting for trigger to be ready. (default 60)
```
